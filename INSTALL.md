# Monokai Pro (CE) for [Zed](https://zed.dev/)

## Installation

### Install via Zed

1. Open Zed.
2. Open the command palette with <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>.
3. Type **zed: extensions**.
4. Search for **Monokai Pro (CE)** and install the extension.

### Manual Installation

1. Clone this repository or download `monokai-pro-ce.json`.
2. Copy `monokai-pro-ce.json` into your Zed theme directory (typically `~/.config/zed/themes`).

### Activating the Theme

1. Open the command palette (<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>).
2. Run **theme selector: toggle**.
3. Choose **Monokai Pro (CE)** from the theme list.
